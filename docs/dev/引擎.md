## Agent 类 (`core/agent.py`)

`Agent` 是整个系统的编排层，持有所有服务实例，管理主循环。它不直接持有上下文状态——`ConversationState` 是唯一的事实源。

### 类签名

```python
class Agent:
    def __init__(
        self,
        enable_log: bool = False,
        resume: str | None = None,
        io: IOChannel | None = None,
    )
```

**参数**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `enable_log` | bool | `False` | 是否输出调试日志 |
| `resume` | str \| None | `None` | 会话恢复模式：`None`/`False`=新会话，`True`/`"auto"`=续最近，`"s_xxx"`=指定会话 |
| `io` | IOChannel \| None | `None` | IO 通道（默认使用 CLI） |

### 核心方法

```python
async def process(self, user_input: str) -> Response
```

处理一条用户输入。返回值 `Response` 包含最终回复内容。内部执行完整的 LLM 调用循环（见架构设计中的数据流图）。

```python
async def shutdown(self)
```

关闭 Agent。触发 `ON_SHUTDOWN` 钩子 → 生成会话摘要 → 保存上下文 → 显示退出面板 → 释放 LLM 客户端连接池。

```python
def reload_skills(self) -> bool
```

热重载所有技能文件。重新构建系统提示词。成功返回 `True`，失败打印错误日志并返回 `False`。

```python
def list_skills(self) -> list[dict]
```

列出所有已加载的技能信息（名称、标题、描述等）。

```python
async def switch_session(self, sid: str) -> bool
```

切换到指定会话。先保存当前上下文，再加载目标会话。

```python
async def clear_session(self)
```

清空当前会话上下文（保留 system prompt）。

```python
def cancel(self)
```

设置取消标记。主循环在下次迭代时检测该标记并中断处理。

### 内置命令

```python
@property
def COMMANDS(self) -> dict[str, str]
```

返回内置命令字典：`{"/exit": "退出", "/clear": "清空会话", ...}`。

```python
async def handle_command(self, cmd_input: str) -> str | None
```

处理以 `/` 开头的命令。返回命令执行结果的文本描述，若命令未知返回 `None`。

### 内置属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `client` | `core.llm_client.Client` | LLM HTTP 客户端实例 |
| `lifecycle` | `LifecycleManager` | 生命周期管理器 |
| `plugins` | `PluginRegistry` | 插件注册表 |
| `session` | `SessionManager` | 会话管理器 |
| `model` | str | 当前模型名称（只读属性，委托给 LLMService） |
| `is_processing` | bool | 是否正在处理请求（只读属性） |

## Response 对象

```python
@dataclass
class Response:
    content: str = ""
    metadata: dict = field(default_factory=dict)
    error: str | None = None
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `content` | str | 最终回复文本（纯文本，非 Markdown） |
| `metadata` | dict | 元数据（当前未使用，预留） |
| `error` | str \| None | 错误信息。非 `None` 时表示处理过程中发生了异常 |

**使用示例**：

```python
agent = Agent(resume="auto")
response = await agent.process("Hello!")
if response.error:
    print(f"Error: {response.error}")
else:
    print(response.content)
```

## LifecycleManager (`core/lifecycle.py`)

事件驱动的生命周期管理器，支持同步/异步钩子，按优先级执行。

### 类签名

```python
class LifecycleManager:
    def __init__(self, enable_log: bool = False)
```

### 核心方法

```python
def register(
    self,
    hook: LifecycleHook,
    func: Callable,
    priority: int = 100,
    name: str | None = None,
    hook_type: str | None = None,
)
```

注册钩子函数。

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `hook` | LifecycleHook | — | 要注册的钩子枚举 |
| `func` | Callable | — | 钩子处理函数（同步或异步均可） |
| `priority` | int | `100` | 优先级，越小越先执行 |
| `name` | str \| None | `None` | 名称（用于注销/调试），默认取函数名 |
| `hook_type` | str \| None | `None` | `"observe"` / `"transform"`。`None`=自动推断 |

```python
def unregister(self, hook: LifecycleHook, name: str) -> bool
```

注销指定钩子。成功返回 `True`，未找到返回 `False`。

```python
async def emit(
    self,
    hook: LifecycleHook,
    context: HookContext | None = None,
    **kwargs,
) -> HookContext
```

异步触发钩子。返回最终的 `HookContext`（可能被 transform 型钩子修改）。调用方应检查返回的 `context.data` 获取插件修改后的值。

```python
def emit_sync(
    self,
    hook: LifecycleHook,
    context: HookContext | None = None,
    **kwargs,
) -> HookContext
```

同步触发钩子（用于非异步环境）。

```python
def get_hooks(self) -> dict[str, list[tuple]]
```

获取所有注册的钩子信息。

```python
def clear(self)
```

清空所有钩子。

```python
def get_stats(self) -> dict[str, int]
```

获取钩子调用统计（每个钩子的调用次数）。

### @hook 装饰器

```python
from core.lifecycle import LifecycleHook, LifecycleManager, hook

lifecycle = LifecycleManager()

@hook(lifecycle, LifecycleHook.ON_BEFORE_LLM_CALL, priority=50)
async def my_llm_guard(context, messages, tools, **kwargs):
    """在 LLM 调用前检查消息内容"""
    for msg in messages:
        if msg["role"] == "user" and len(msg["content"]) > 10000:
            context.data["cancelled"] = True
            context.data["cancel_reason"] = "消息过长"
    return context
```

**装饰器定义**（示意）：

```python
def hook(lifecycle: LifecycleManager, hook_enum: LifecycleHook, priority: int = 100, name: str | None = None):
    def decorator(func):
        lifecycle.register(hook_enum, func, priority=priority, name=name)
        return func
    return decorator
```

### 完整示例

```python
from core.lifecycle import LifecycleManager, LifecycleHook, HookContext

lifecycle = LifecycleManager(enable_log=True)

# observe 型钩子：仅观察，不修改
def on_message(ctx: HookContext, **kwargs):
    print(f"[Observe] 用户说了: {kwargs.get('filtered_content', '')[:50]}")

lifecycle.register(LifecycleHook.ON_MESSAGE_RECEIVED, on_message, priority=200)

# transform 型钩子：可修改数据
async def on_filter(ctx: HookContext, **kwargs):
    content = kwargs.get("filtered_content", "")
    if "敏感词" in content:
        ctx.data["blocked"] = True
        ctx.data["block_reason"] = "包含敏感内容"
    return ctx

lifecycle.register(LifecycleHook.ON_MESSAGE_FILTER, on_filter, priority=50)
```
