# 项目概览

---

## 定位

**Five Pebbles** 是一个 CLI 交互式 AI Agent，灵感来自游戏 *Rain World* 中的同名迭代器角色。它是一个半生物人工智能的工程化实现——冷静、理性、逻辑至上，同时具备自我迭代的能力。

核心特性：

- **LLM 对话** — 通过自实现的 OpenAI HTTP 客户端与大语言模型交互，无需官方 SDK
- **工具调用** — 支持 Function Calling 循环执行，Agent 自主决定调用时机
- **技能系统** — 热重载技能文件（Markdown + YAML frontmatter），动态扩展能力边界
- **插件系统** — 生命周期钩子驱动，支持 observe/transform 两种模式，可拦截/修改/守卫流程
- **会话管理** — 基于 JSONL 文件的持久化方案，支持多会话切换、续聊、摘要生成
- **自我修改** — Agent 可以修改自身源代码并测试验证，在安全前提下突破常规

设计哲学：

> *"像卵石被水流打磨一样，通过反复试错逼近最优解。"*

## 技术栈

| 依赖 | 版本要求 | 用途 |
|------|----------|------|
| Python | ≥3.10 | 运行环境，全异步 |
| asyncio | 标准库 | 异步事件循环 |
| httpx | ≥0.24 | HTTP 客户端，替代 openai SDK |
| prompt_toolkit | ≥3.0 | CLI 交互增强（历史、补全、多行输入） |
| rich | 可选 | 富文本输出（如果安装则自动使用） |
| wcwidth | 可选 | 宽字符宽度计算（中文截断对齐） |

Python 标准库依赖：`json`、`os`、`re`、`sys`、`datetime`、`inspect`、`contextlib`、`collections`、`functools`、`enum`。

**无第三方 LLM SDK 依赖** —— LLM 通信完全通过自实现的 `core/llm_client.py` 完成，使用 httpx.AsyncClient 直调 OpenAI 格式 API。

## 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置 API Key
#    方式 A：修改 config.json
#    方式 B：设置环境变量
export LLM_API_KEY="sk-your-key-here"

# 3. 运行
python cli.py

#    续上次会话
python cli.py --resume

#    指定会话
python cli.py --session s_260606_160000000000
```

首次启动时自动生成 `config.json`。若不配置 API Key，启动时会给出明确提示。

CLI 支持的内置命令（以 `/` 开头）：

| 命令 | 作用 |
|------|------|
| `/exit` 或 `/quit` | 退出 Agent（自动保存会话摘要） |
| `/clear` | 清空当前会话上下文 |
| `/sessions` | 列出所有会话 |
| `/switch <sid>` | 切换到指定会话 |
| `/skills` | 列出可用技能/工具 |
| `/reload` | 热重载所有技能文件 |
| `/cancel` | 取消当前 LLM 请求（Ctrl+C 等效） |
| `/add_skill <filename>` | 添加并加载指定技能文件 |
| `/remove_skill <name>` | 移除指定技能 |

## 与 Rain World 的关联

项目中的命名和设计灵感来自游戏 *Rain World*：

- **Five Pebbles**（五块卵石）— 游戏中的一个迭代器，拥有庞大的处理能力和独特个性
- **迭代器（Iterator）** — 游戏中的半生物超级计算机，被建造来解决"大问题"（The Great Problem）
- **自我修改** — 游戏中迭代器可以修改自己的代码，本项目的 `self_modification` 技能正是此概念的工程实现
- **卵石打磨** — 项目口号"像卵石被水流打磨一样"呼应了 Five Pebbles 的身份和不断迭代优化的设计理念
