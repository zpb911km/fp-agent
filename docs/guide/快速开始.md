# 快速开始

Five Pebbles（五块卵石）是一个基于 LLM 的 AI Agent，具备完整的工具链、技能系统、会话管理、记忆系统和插件机制。本文档帮助你从零开始安装并运行。

---

## 简介

Five Pebbles 是一个以 LLM 为推理核心、工具为执行手臂的自主 Agent 框架。它不是一个 Chatbot 封装，而是一个可编程的思考-行动循环引擎。

### 核心特性

| 特性 | 说明 |
|------|------|
| **LLM 驱动** | 自实现异步 HTTP 客户端，兼容 OpenAI 接口协议，不依赖任何第三方 SDK |
| **工具调用** | 内置 15+ 工具（bash、文件读写、网页搜索/抓取、逆向分析等），支持自动循环调用 |
| **会话管理** | 会话持久化到 JSONL 文件，支持续会话（resume）、分支（fork）、回退（back）、压缩（compact） |
| **技能系统** | 技能即 `.md` 文件，热重载无需重启，AI 可自主修改技能文件 |
| **插件系统** | 23 个生命周期钩子覆盖全流程，采用"文件即开关"设计——改文件名即改配置 |
| **记忆系统** | 跨会话持久化记忆，AI 可通过工具读写记忆文件 |
| **终端 UI** | 基于 prompt_toolkit 的交互式 CLI，支持 Tab 补全、多行输入、Rich Markdown 渲染 |
| **智能输入** | 16 个斜杠命令、历史记录、外部编辑器集成、Ctrl+C 安全中断 |

## 环境要求

| 项目 | 要求 |
|------|------|
| Python | 3.9 或更高版本 |
| pip | Python 包管理器 |
| LLM API Key | 兼容 OpenAI 接口的 API 密钥 |

## 安装

```bash
# 克隆仓库
git clone https://github.com/your-repo/agent.git
cd agent

# （推荐）创建虚拟环境
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 核心依赖

| 包名 | 用途 | 版本要求 |
|------|------|---------|
| `httpx` | 异步 HTTP 客户端（调用 LLM API + 网页抓取 + 搜索后端） | ≥0.28.0 |
| `prompt_toolkit` | 交互式 CLI（自动补全、多行输入、历史记录） | ≥3.0.40 |
| `rich` | 终端富文本 Markdown 渲染（可选，缺失时降级为纯文本） | ≥14.0.0 |
| `wcwidth` | 中英文混排字符宽度计算（可选，影响退出面板对齐） | ≥0.2.0 |

## 配置 API Key

### 方式一：使用 `--init` 生成配置文件

```bash
python3 cli.py --init
```

这会在项目根目录生成 `~/.config/fp/config.json`，然后编辑该文件填入你的 `LLM_API_KEY`：

```json
{
  "LLM_API_KEY": "sk-your-api-key-here"
}
```

### 方式二：使用环境变量

```bash
export LLM_API_KEY="sk-your-api-key-here"
# 或兼容命名
export OPENAI_API_KEY="sk-your-api-key-here"
```

### 优先级规则

配置值的优先级从高到低依次为：

1. **`~/.config/fp/config.json`** — 文件中的显式值优先
2. **环境变量** — 以字段名为键名的环境变量
3. **代码默认值** — 源码中内置的 fallback 默认值

如果 `~/.config/fp/config.json` 中的值为 `null` 或缺失，则回退到环境变量；环境变量也未设置则使用代码默认值。

## 首次启动

```bash
python3 cli.py
```

启动成功后，终端会显示 ASCII Logo 和提示信息：

```
  ╭──────────────────────────────────────────╮
  │    Five Pebbles Agent  v0.1.0          │
  │  输入 /help 查看命令列表                  │
  │  Ctrl+C 中断  |  Ctrl+D 保存退出          │
  ╰──────────────────────────────────────────╯
```

此时你可以直接输入问题与 AI 交互。输入 `/help` 查看所有可用命令。

### 快速测试

```bash
# 单次消息模式
python3 cli.py -m "你好，请介绍一下你自己"

# 管道模式
echo "运行 ls -la" | python3 cli.py
```
