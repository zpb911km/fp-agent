# 命令系统 —— 导入、注册与覆写机制（全异步重构版）

命令系统提供用户与 Agent 交互的文本界面入口：输入 `/command arg` 触发对应操作。

---

## 1. 命令模块接口

每个命令是一个独立的 `.py` 文件，必须导出以下接口：

| 导出名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `name` | `str` | 是 | 命令名（不含 `/`） |
| `execute` | `(agent, arg: str) -> tuple[bool, str]` | 是 | 执行函数 |
| `description` | `str` | 否 | 帮助描述（用于 `/help`） |
| `aliases` | `list[str]` | 否 | 别名列表 |

### execute 返回值兼容设计（三层）

`execute` 函数支持三种返回值形态，调度器自动适配：

| 返回值类型 | 含义 | 示例场景 |
|---|---|---|
| `bool` | 旧版兼容，自动补为 `(bool, "")` | `exit_bang.py`（仅需控制循环） |
| `tuple[bool, str]` | 标准新版：`(是否已处理, 输出文本)` | `help.py`, `clear.py` |
| `tuple[bool, str, str]` | 扩展预留（当前未显式使用） | 未来兼容 |

`execute` 可以是同步函数也可以是 `async def`，调度器通过 `inspect.iscoroutinefunction` 自动检测并适配调用方式。

### 完整示例

```python
"""commands/hello.py"""
name = "hello"
description = "向用户打招呼"
aliases = ["hi", "hey"]

def execute(agent, arg: str) -> tuple[bool, str]:
    target = arg.strip() or "World"
    msg = f"Hello, {target}!"
    return (True, msg)
```

## 2. 自动发现机制（`_discover_commands`）

入口在 `commands/__init__.py`，模块导入时自动执行 `_discover_commands()`。

### 2.1 双层扫描流程

```
_discover_commands()
  ├── 清空 _commands 全局 dict
  ├── _scan_dir(builtin_dir, "fp_core.commands")   # ① 内置命令（包导入）
  └── _scan_dir(user_dir)                             # ② 用户命令（文件路径导入）
```

| 维度 | 内置命令（built-in） | 用户命令（user） |
|------|----------------------|------------------|
| 导入方式 | `importlib.import_module()`（包导入） | `importlib.util.spec_from_file_location()`（文件路径导入） |
| 包前缀 | `fp_core.commands` | 无 |
| 覆盖优先级 | 低（先加载） | **高**（后加载，同名覆盖） |
| 扫描目录 | `fp_core/commands/` | `~/.local/share/fp/commands/` |

### 2.2 核心实现逻辑（伪码示意）

```python
# commands/__init__.py

_commands: dict[str, ModuleType] = {}

def _scan_dir(directory: str, package_prefix: str | None = None) -> None:
    """扫描目录下全部 .py 文件（排除 __init__.py），动态导入并注册"""
    for entry in os.scandir(directory):
        if not entry.name.endswith(".py") or entry.name == "__init__.py":
            continue
        modname = entry.name[:-3]
        if package_prefix:
            # 内置命令：包导入
            mod = importlib.import_module(f"{package_prefix}.{modname}")
        else:
            # 用户命令：文件路径导入
            spec = importlib.util.spec_from_file_location(modname, entry.path)
            mod = importlib.util.module_from_spec(spec)
            spec.loader.exec_module(mod)
        _register_module(mod)

def _register_module(mod: ModuleType) -> None:
    """校验接口并注册到 _commands 字典"""
    if not hasattr(mod, "name") or not hasattr(mod, "execute"):
        return  # 静默跳过不合法模块
    name = mod.name
    if name in _commands:
        display.warning(f"⚠️ 命令 [{name}] 重复定义，已覆盖")
    _commands[name] = mod
    # 注册别名
    for alias in getattr(mod, "aliases", []):
        if alias in _commands:
            display.warning(f"⚠️ 别名 [{alias}] 冲突，已跳过")
            continue
        _commands[alias] = mod

_discover_commands()  # 模块加载时自动执行
```

## 3. 同名覆写规则

| 场景 | 行为 | 是否告警 |
|------|------|----------|
| 内置命令先加载，用户命令同名后加载 | **覆盖**，用户版本生效 | ✅ 打印 warning |
| 用户命令之间同名 | **覆盖**，后加载者胜出 | ✅ 打印 warning |
| 别名与已有主名/别名冲突 | **跳过**，保留原条目 | ✅ 打印 warning |

**关键差异**：主名采用**覆盖**策略（后者胜出），别名采用**跳过**策略（保留先注册者）。这确保了用户命令可以替换内置命令，但别名不会意外覆盖已有命令。

## 4. 命令执行流程

```
agent._process_inner()
  └─ agent.handle_command(cmd_line)          # 解析 "/cmd arg" → (cmd_name, arg)
       └─ commands.execute(self, cmd_name, arg)
            ├─ get_command(cmd_name)          # _commands 字典 O(1) 查找
            ├─ iscoroutinefunction?           # await / 直接调用
            └─ 返回 (handled, output)
```

### 4.1 调度器适配逻辑

```python
async def execute(self, agent, cmd_name: str, arg: str) -> tuple[bool, str]:
    mod = _commands.get(cmd_name)
    if mod is None:
        return (False, f"未知命令: {cmd_name}")

    if inspect.iscoroutinefunction(mod.execute):
        return await mod.execute(agent, arg)
    else:
        result = mod.execute(agent, arg)
        # 返回值归一化：兼容 bool / tuple[bool, str] / tuple[bool, str, str]
        return _normalize_result(result)
```

### 4.2 返回值归一化

```python
def _normalize_result(result) -> tuple[bool, str]:
    if isinstance(result, tuple):
        if len(result) == 2:
            return result          # (bool, str)
        elif len(result) == 3:
            return (result[0], result[1])  # (bool, str, str) → 截取前两项
    return (result, "")  # bool → 自动补空字符串
```

## 5. 用户命令编写指南

1. **创建文件**：在 `~/.local/share/fp/commands/` 下创建 `.py` 文件
2. **最小导出**：至少导出 `name` 和 `execute`
3. **避免相对导入**：用户命令通过文件路径加载，不支持 `from .xxx`，建议自包含单文件脚本
4. **覆写内置命令**：同名即可自动覆盖，无需额外配置
5. **异步支持**：`execute` 可声明为 `async def`，调度器会自动 `await`

## 7. 架构全景图（数据流）

```
┌──────────────────────────────────────────────┐
│              agent._process_inner()           │
│  ┌───────────┐   ┌──────────────────────┐    │
│  │ 用户输入   │ → │ handle_command()      │    │
│  │ "/cmd arg" │   │ 解析 cmd_name + arg   │    │
│  └───────────┘   └──────────┬───────────┘    │
│                             ▼                 │
│  ┌─────────────────────────────────────────┐  │
│  │        commands.execute()               │  │
│  │  ┌──────────────┐  ┌─────────────────┐ │  │
│  │  │ O(1) 查找    │  │ 同步/异步自动适配│ │  │
│  │  │ _commands    │  │ iscoroutinefunction│ │  │
│  │  └──────────────┘  └─────────────────┘ │  │
│  └─────────────────────────────────────────┘  │
│                       ▼                       │
│  ┌─────────────────────────────────────────┐  │
│  │     返回值归一化 _normalize_result()     │  │
│  │   bool → (bool, "")                      │  │
│  │   (bool, str) → 保持不变                 │  │
│  │   (bool, str, str) → 截取前两项          │  │
│  └─────────────────────────────────────────┘  │
└──────────────────────────────────────────────┘
        │
        ▼
  (handled, output) → agent 处理展示
```
