# 配置指南

Five Pebbles 通过 `config.json` 文件进行配置，同时支持环境变量覆盖。所有配置项在启动时按三级优先级加载。

---

## config.json 完整字段

```json
{
  "LLM_API_KEY": "sk-your-api-key-here",
  "LLM_API_BASE_URL": "https://api.deepseek.com/v1",
  "LLM_MODEL": "deepseek-v4-flash",
  "TEMPERATURE": 0.8,
  "MAX_TOKENS": 32768,
  "TIMEOUT": 300,
  "RETRY_COUNT": 3,
  "MAX_ITERATIONS": 50,
  "MAX_CONTEXT_TOKENS": 8000,
  "MEMORY_MAX_HISTORY": 100,
  "display_styles": {
    "info": { "color": "green" },
    "error": { "color": "bright_red", "bold": true },
    "warning": { "color": "yellow" },
    "yellow_bold": { "color": "yellow", "bold": true },
    "hint": { "color": "magenta" },
    "llm_thought": { "color": "magenta", "dim": true },
    "llm_tool": { "color": "yellow" },
    "startup": { "color": "green" },
    "logo": { "color": "magenta" }
  },
  "display_truncation": {
    "llm_thought": 100,
    "llm_tool": 300,
    "logo": -1
  }
}
```

---

## 字段详解

### LLM 连接配置

| 字段 | 必填 | 类型 | 默认值 | 说明 |
|------|------|------|--------|------|
| `LLM_API_KEY` | 是 | string | `""` | LLM 服务的 API 密钥，兼容任何 OpenAI 接口协议的提供商 |
| `LLM_API_BASE_URL` | 否 | string | `"https://api.deepseek.com/v1"` | API 基础地址，换用其他服务商时修改此处 |
| `LLM_MODEL` | 否 | string | `"deepseek-v4-flash"` | 模型名称，需与 API 提供商支持的模型名一致 |
| `TEMPERATURE` | 否 | float | `0.8` | 生成温度，值越高输出越随机。范围 0.0 ~ 2.0 |
| `MAX_TOKENS` | 否 | int | `32768` | 单次 LLM 调用的最大输出 Token 数 |
| `TIMEOUT` | 否 | int | `300` | HTTP 请求超时时间（秒） |
| `RETRY_COUNT` | 否 | int | `3` | LLM 调用失败时的重试次数 |

### Agent 行为配置

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `MAX_ITERATIONS` | int | `50` | 单次用户消息的最大工具调用迭代次数，超过则强制跳出防止无限循环 |
| `MAX_CONTEXT_TOKENS` | int | `8000` | 上下文窗口阈值（Token 数），超过时启动压缩策略 |
| `MEMORY_MAX_HISTORY` | int | `100` | 记忆系统保留的最大历史条目数 |

---

## 显示配置

### display_styles

每个样式名对应一个颜色和字体效果对象。可用颜色名称：`black`、`red`、`green`、`yellow`、`blue`、`magenta`、`cyan`、`white`、`bright_red`、`bright_green`、`bright_yellow`、`bright_cyan`、`bright_white`、`default`。字体效果支持 `bold`（粗体）、`dim`（暗淡）、`italic`（斜体）。

| 样式名 | 默认颜色 | 效果 | 用途 |
|--------|---------|------|------|
| `info` | green | 普通 | 一般信息提示、操作成功反馈 |
| `error` | bright_red | bold | 错误消息 |
| `warning` | yellow | 普通 | 警告信息 |
| `yellow_bold` | yellow | bold | 强调性警告，如中断标记 |
| `hint` | magenta | 普通 | 行为引导提示 |
| `llm_thought` | magenta | dim | LLM 思考过程的灰色显示 |
| `llm_tool` | yellow | 普通 | 工具调用和结果的显示 |
| `startup` | green | 普通 | 启动横幅 |
| `logo` | magenta | 普通 | ASCII Logo 艺术字 |

### display_truncation

控制 LLM 流输出在终端中的截断策略。值为 `-1` 表示不截断。

| 截断名 | 默认值 | 说明 |
|--------|--------|------|
| `llm_thought` | `100` | LLM 思考过程的截断长度（字符数），超出部分以 `… <+N chars>` 省略 |
| `llm_tool` | `300` | 工具调用及结果的截断长度（字符数） |
| `logo` | `-1` | 启动 Logo 不截断（始终完整显示） |
