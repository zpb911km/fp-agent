# 命令系统

命令系统提供用户与 Agent 交互的文本界面入口：输入 `/command arg` 触发对应操作。

---

## 命令文件格式

每个命令是一个独立的 Python 模块，必须导出以下接口：

| 导出名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `name` | `str` | 是 | 命令名（不含 `/`） |
| `description` | `str` | 是 | 简短帮助描述 |
| `execute` | `(agent, arg: str) -> bool` | 是 | 执行函数；返回 `True` 继续循环，`False` 退出 |
| `aliases` | `list[str]` | 否 | 别名列表 |

**完整示例：**

```python
"""commands/hello.py — 示例命令"""
name = "hello"
description = "向用户打招呼"
aliases = ["hi", "hey"]

def execute(agent, arg: str) -> bool:
    """arg 是用户输入的剩余参数（不含命令名）"""
    target = arg.strip() or "World"
    agent.display.info(f"Hello, {target}!")
    return True  # 继续主循环
```

---

## 异步命令

`execute` 可以是 `async def`，系统自动检测并适配：

```python
async def execute(agent, arg: str) -> bool:
    result = await some_async_operation()
    agent.display.info(result)
    return True
```

命令调度器通过 `inspect.iscoroutinefunction(execute)` 判断，是则 `await`。

---

## 自动发现机制

`commands/__init__.py` 扫描 `fp_core/commands/` 目录下的全部 `.py` 文件（排除 `__init__.py`），动态构建 **命令名 → 模块** 映射，含别名索引。

**核心扫描逻辑（伪代码）：**

```python
# commands/__init__.py
import importlib, pkgutil, inspect
import commands as pkg

COMMAND_MAP = {}       # name → module
ALIAS_MAP = {}         # alias → name

for _, modname, _ in pkgutil.iter_modules(pkg.__path__):
    if modname == "__init__":
        continue
    mod = importlib.import_module(f"commands.{modname}")
    # 接口校验
    if not hasattr(mod, "name") or not hasattr(mod, "execute"):
        continue
    name = mod.name
    COMMAND_MAP[name] = mod
    for alias in getattr(mod, "aliases", []):
        ALIAS_MAP[alias] = name

def dispatch(agent, raw: str) -> bool:
    # raw 形如 "/command arg"
    parts = raw[1:].strip().split(maxsplit=1)
    cmd_name = parts[0]
    arg = parts[1] if len(parts) > 1 else ""
    mod = COMMAND_MAP.get(cmd_name) or COMMAND_MAP.get(ALIAS_MAP.get(cmd_name))
    if mod is None:
        return True  # 未知命令静默忽略
    if inspect.iscoroutinefunction(mod.execute):
        import asyncio; return asyncio.run(mod.execute(agent, arg))
    return mod.execute(agent, arg)
```

| `model` | — | 查看/切换 LLM 模型 |
| `history` | — | 查看对话历史 |
| `back` | — | 回退到上一个分支点 |
| `reset` | — | 重置当前会话上下文 |
| `clear` | — | 清屏 |
| `resume` | — | 恢复被中断的对话 |
| `skills` | — | 列出已加载的所有技能 |
| `reload_skills` | — | 热重载全部技能文件 |
| `remove_skill` | — | 移除指定技能 |
| `fork` | — | 从当前上下文分叉为新会话 |
| `compact` | — | 压缩会话（摘要旧消息） |
| `memory` | — | 查看/管理长期记忆 |
| `exit` | `quit` | 正常退出（保存会话） |
| `exit!` | — | 强制退出（不保存） |
