# 技能系统

技能是注入到 LLM 系统提示词中的知识/能力模块。每个技能是一个 Markdown 文件，通过 YAML frontmatter 定义元数据，正文为技能描述文本。

---

## SkillLoader (skills/loader.py)

### 数据模型

```python
from dataclasses import dataclass, field
from typing import Optional

@dataclass
class Skill:
    name: str                     # 唯一标识名
    title: str                    # 展示标题
    description: str              # 简要描述
    content: str                  # 技能正文（Markdown）
    category: str = "general"     # 分类
    version: str = "1.0.0"        # 版本号
    priority: int = 100           # 优先级（值越小排序越前）
    metadata: dict = field(default_factory=dict)  # 扩展元数据
```

### 加载器 API

```python
class SkillLoader:
    def load_all(self) -> list[Skill]:              # 扫描并加载所有技能
    def get_skill(self, name: str) -> Skill | None: # 按名称获取单个技能
    def get_all_prompt_text(self) -> str:           # 拼接所有技能为提示词文本
    def reload(self):                                # 热重载全部
```

### 全局单例

`skills/loader.py` 底部创建全局实例：

```python
skill_loader = SkillLoader()
```

其他模块通过 `from fp_core.skills.loader import skill_loader` 访问。

## 技能文件格式

文件位于 `fp_core/skills/` 目录下，扩展名 `.md`。格式为 **YAML frontmatter + Markdown 正文**。

```markdown
---
name: web_search
title: 网络搜索专家
description: 通过搜索获取最新网络信息
category: tool
version: 1.2.0
priority: 10
---

## 网络搜索技能

当需要获取实时或最新的信息时，使用 web_search 工具搜索网络。
```

**必填字段：** `name`、`title`、`description`

**可选字段：** `category`（默认 `"general"`）、`version`（默认 `"1.0.0"`）、`priority`（默认 `100`）

---

## 加载流程

### `load_all()` 扫描逻辑

1. 调用 `glob("skills/*.md")` 获取所有技能文件
2. 对每个文件：
   - 用 `re.match(r'^---\s*\n(.*?)\n---\s*\n', content, re.DOTALL)` 提取 frontmatter
   - `yaml.safe_load()` 解析元数据
   - 剩余内容为 `content`
   - 构造 `Skill` 对象

### `get_all_prompt_text()` 拼接逻辑

按 `priority` **降序**（高优先级在前）逐条拼接：

```
## {skill.title}

{skill.content}
```

---

## 无 frontmatter 降级

若文件没有有效的 YAML frontmatter，自动降级处理：

| 字段 | 推导规则 |
|------|----------|
| `name` | 文件名（不含 `.md`） |
| `title` | 文件名，下划线转空格 + `.title()` |
| `description` | 正文前 200 个字符（去除首尾空白） |
| `content` | 全文 |
| `category` | `"general"` |
| `priority` | `100` |

例：`skills/my_custom_tool.md` → `name="my_custom_tool"`, `title="My Custom Tool"`。

## 热重载

`SkillLoader.reload()` 清空内部缓存后重新执行 `load_all()`。

```python
def reload(self):
    self._skills = []
    self._skills_map = {}
    self.load_all()
```

`Agent.reload_skills()` 调用 `skill_loader.reload()` 并重新构建 system prompt。
