# 技能系统

## 技能文件位置

所有技能文件存储在 `fp_core/skills/` 目录下，每个技能一个独立文件。

```bash
ls -lh skills/
```

---

## 技能文件格式

技能文件使用 **YAML frontmatter + Markdown 正文** 的格式。

**必填字段**

| 字段 | 说明 |
|------|------|
| `name` | 技能唯一标识名（英文短词，如 `file_fingerprint`） |
| `title` | 技能显示标题（如「文件指纹识别」） |
| `description` | 技能功能描述（一句话） |

**可选字段**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| `category` | 分类标签 | `general` |
| `version` | 版本号 | `1.0` |
| `priority` | 优先级（数值越小越优先加载） | `100` |

**示例**

```markdown
---
name: file_fingerprint
title: 文件指纹识别
description: 使用 file 和 strings 命令提取特征，判断类型
category: analysis
version: 1.0
priority: 50
---

使用 `file` 和 `strings` 命令提取二进制文件特征。

**使用场景**
- 遇到未知二进制文件时，先执行指纹识别
- 快速判断文件类型，无需打开

**调用方式**
```python
file_fingerprint(file_path="/path/to/file")
```
```

## 管理技能

### 列出技能

`/skills` 命令列出所有已加载的技能及其状态。

```
> /skills
📦 已加载技能（13 个）

  file_fingerprint      🟢 文件指纹识别
  process_tracking      🟢 进程行为动态追踪
  text_processing       🟢 高级文本处理
  elf_analysis          🟢 ELF 文件分析
  privilege_escalation  🟢 权限提升扫描
  env_pollution_check   🟢 环境变量污染检测
  yt_dlp_download       🟢 视频下载
  python_syntax_check   🟢 Python 语法检查
  lanqiao_fetch         🟢 蓝桥杯题目抓取
  lanqiao_submit        🟢 蓝桥杯代码提交判题
  modify_skills         🟢 添加修改删除技能
  delete_empty_sessions 🟢 清理空会话
  self_modification     🟢 修改自身源代码
```

### 热重载

编辑技能文件后，执行 `/reload_skills` 即可热重载，无需重启程序。

```
> /reload_skills
🔄 技能已重新加载（13 个）
```

### 移除技能

使用 `/remove_skill` 命令移除指定技能。

```
> /remove_skill file_fingerprint
🗑️ 技能 file_fingerprint 已移除
```

## 内置技能一览

系统内置 13 个技能，按功能分类：

**分析类**
| 技能名 | 说明 |
|--------|------|
| `file_fingerprint` | 文件指纹识别（file + strings 命令） |
| `process_tracking` | 进程行为动态追踪（系统调用/网络活动） |
| `text_processing` | 高级文本处理（awk/sed/rg 模式匹配） |
| `elf_analysis` | ELF 文件结构分析 |
| `privilege_escalation_scan` | 权限提升线索扫描（SUID/SGID） |
| `env_pollution_check` | 环境变量污染检测 |

**下载类**
| 技能名 | 说明 |
|--------|------|
| `yt_dlp_download` | 从 YouTube 等网站下载音视频 |

**代码类**
| 技能名 | 说明 |
|--------|------|
| `python_syntax_check` | Python 语法检查（ast 模块） |
| `lanqiao_fetch` | 蓝桥杯题目抓取 |
| `lanqiao_submit` | 蓝桥杯代码提交判题 |

**系统类**
| 技能名 | 说明 |
|--------|------|
| `modify_skills` | 添加/修改/删除技能文件 |
| `delete_empty_sessions` | 扫描并清理空会话文件 |
| `self_modification` | 修改自身源代码并测试验证 |
